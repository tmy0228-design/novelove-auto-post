# Novelove 自動投稿システム (v21.0.0)

> [!IMPORTANT]
> **開発・修正時の注意**: 作業を開始する前に、必ず `git pull origin main` で最新版をプルしてから作業を行ってください。
> 
> **【🔥コアコンセプト】**
> 本システムの最大の目的は**「DLsite/FANZA/らぶカル/DMMの新作作品を『最速』で紹介すること」**です。口コミやレビューがない発売前の段階で、少ない情報からいかに魅力的な紹介記事を自動生成するかが肝となります。将来のAIはこの前提を忘れた提案（レビュー取得等）をしてはいけません。
> ※ DigiKetはv20.0.4で関連コード・定義を完全削除。

Novelove（ノベラブ）のコンテンツ生成および自動投稿を行うエンジン一式です。
BL・TL の女性向け作品（漫画・小説・同人）を全自動で紹介記事化して WordPress に投稿します。

## 🚀 システム概要
- **目的**: DLsite, DMM.com(一般), らぶカル(Lovecal) の新着・ランキングを取得し、**DeepSeek V4-Flash**（直接API / OpenRouterフォールバック）で紹介記事を生成して WordPress に自動投稿する。※DigiKetはv20.0.4で関連コード・定義を完全削除。
- **主要言語**: Python 3.x
- **データベース**: SQLite (`novelove_unified.db`)

## ✨ 主な機能 (v20.1.0)

### 0. らぶカル(LoveCal) 完全統合 (v13.6.0)
FANZAの同人新フロア「らぶカル」のBL/TLコンテンツを既存パイプラインに完全統合（2025年3月3日サイト分離）。
- スクレイピング・フィルタリング・セール検知・ダッシュボード表示が全て自動対応
- 投稿時には「FANZA」ではなく「らぶカル」独立タグを自動付与（SEO用Description設定済み）

### 1. 2名掛け合いランキング記事
毎週末のランキング記事をMC＋ゲストの**掛け合い（吹き出し対話）形式**で生成。
5名×4名=全10通りの関係性を `novelove_soul.py` の `RELATIONSHIPS` 辞書で定義し、プロンプトに自動注入します。

### 2. AI SEOメタ自動生成・付与
通常・ランキングともに、DeepSeek V4-Flashが記事本文をベースに強力なSEO特化のテキストを生成。
- **seo_title**: 『作品名』＋AIが考えた感情を揺さぶる惹き句（68文字以内）
- **meta_desc**: あらすじの単なる抜粋ではなく、誰のどんな性癖に刺さるかを端的に伝える紹介文（80文字程度）

### 3. HTML骨格パターンの自律多様化 (A/B/C/D)
Googleの「Scaled Content Abuse（テンプレート量産）」ペナルティを防ぐため、記事のDOM構造そのものをAIがランダムに切り替えます。
- 従来の箇条書きベースの「リスト型」から、`ul/li`を禁止して見出しで深掘りする「深掘り型」、読者の疑問に答える「Q&A型」、あらすじ原文を引用する「引用型」まで、情報量に応じて最適なHTML構造を動的に組み立てます。

### 4. 多段ガードレールと関所（Cooldown）システム
- **サーキットブレーカー**: 連続エラー3回 or 5分超過で自動停止 → Discord緊急通知（`emergency_stop.lock`生成）
- **関所（冷却機能）**: SQLiteのロックエラーを防ぐため、cron多重起動時の安全弁として冷却ロジックを実装。
- **クールダウン**: 通常投稿は35分待機（cron 20分間隔との組み合わせでタイミングズレのない実効40分間隔≈約36件/日）。ランキング記事は週１回の投稿を厳密に保証するため、「その週の対象データベースに完了記録(slug)があるか」を直接照合し、スキップミスを100%防止。
- **重複防止**: DB全件 + WordPressの公開記事タイトルとの照合により二重投稿を100%排除し、SQLite固有の `ORDER BY 0` エラーも防ぐ堅牢なSQL構造。

### 5. DLsite一般同人（全年齢）自動取得 (v20.0.0)
DLsiteの一般向け同人作品（全年齢）を自動取得対象に追加。BL/TLに加え、全年齢カテゴリの新着も自動で記事化・投稿します。

### 6. Bluesky自動投稿連携 (v20.0.0)
記事公開時にBlueskyへ自動連携投稿。`novelove_bluesky.py` モジュールにより、WordPress投稿完了後にBlueskyへ記事リンク付きポストを自動送信します。

### 7. テーマカスタマイズ (v20.0.1)
- **ナビゲーターV3フィルター**: AIライター（ナビゲーター）の表示をV3仕様のフィルターで制御
- **JSシャッフル**: フロントエンド表示順のJavaScriptシャッフル機能
- **bcacheパージ**: テーマ更新時のブラウザキャッシュ自動パージ対応

### 8. らぶカル source_db 修正とDigiKet完全削除 (v20.0.4)
- **source_db 不一致の修正**: 旧FANZA商業除外の影響でらぶカル投稿が停止していた不整合を修正し、ラウンドロビンの稼働を回復。
- **DMM/らぶカル関数名整理**: コード内の fetch_fanza_* 関数群を実態に合わせて fetch_dmm_* へリネームし、不要な DigiKet 等のインポートを整理。
- **DigiKet完全クリーンアップ**: WordPress上から過去記事を完全削除したことに伴い、これまで互換維持していたアフィURL生成、スクレイピング、環境変数・定数定義などの全関連コードを安全に完全撤去。

### 9. 話数・巻数違いの誤排除防止 (v20.0.6 - v20.0.7)
- **末尾数字・巻数判定**: タイトル末尾の数字（括弧なし・丸数字含む）や「話」「巻」表記を検出し、話数・巻数違いの類似タイトルを重複と誤判定してスキップする問題を解消。
- **あらすじ類似度比較**: タイトルの類似度が高くても、あらすじの類似度が低い場合は「巻数違いの別作品」とみなす二段階比較セーフガードを追加。

### 10. 2ページ目以降の noindex 設定の解除 (v20.0.8)
- **インデックス化推進**: カテゴリ、タグ、その他アーカイブの2ページ目以降の noindex 設定をデータベースから完全解除し、検索エンジン巡回を有効化。

### 11. 専売・セール・売れ筋のマルチステータス自動同期 (v20.0.9)
- **プレフィックス＆バナー統合**: タイトルプレフィックスの表示制限（最大2つ）、バナーの排他統合ロジックを実装し、表示の美しさとSEO訴求を両立。

### 12. AIタグホワイトリスト大規模整理とメタ設定 (v20.1.0)
- **トレンド追加と重複排除**: トレンドタグ（ざまぁ、悪役令嬢等）を追加し、重複タグ（契約結婚等）を統合。新規・既存タグのCocoon SEOメタ情報を拡充。

### 13. テーマ別まとめ記事の自動生成 (v21.0.0)
- **週次自動オートパイロット**: BL / TL / cross（クロスタグ）の自動選定と月4本投稿のスケジュール管理。
- **クールダウン機能**: 過去90日間に使用されたタグ（クロスタグは要素に分解）を除外して重複投稿を完全防止。
- **安全性とSEO**: あらすじ情報に基づく特化レビュー（方法C）の分割生成と、比較テーブル、タグアーカイブ・最新ランキング記事へのフッターリンク自動付与。
